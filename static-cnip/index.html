关于

简洁 · 快速 · 面向开发者的 IP 查询工具

cnip.io 是一个面向中文用户的 IP 查询工具，专注于提供快速、准确、稳定的 IP 归属地与网络信息解析服务。适用于开发调试、网络分析与日常查询场景。

数据来源

基于 ip2region 高级离线数据库，支持 IPv4 / IPv6 双栈查询。

  • 国家 / 省份 / 城市
  • 运营商 (ISP) / ASN
  • 经纬度 / 时区
  • 17+ 核心字段覆盖

自动更新

数据库定时自动同步，持续跟踪最新 IP 分配信息。

  • 定时调度，无需人工干预
  • 热更新，服务不中断
  • 确保查询结果时效性

隐私保护

我们重视用户隐私，承诺以下原则：

  • 不记录任何查询日志
  • 不进行用户行为追踪
  • 请求即时处理后丢弃
